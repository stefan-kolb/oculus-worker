# oculus-worker

Ruby crawler to track language runtimes.

Currently supported runtimes are:
- Clojure
- Dart
- Dotnet
- Erlang
- Go
- Groovy
- Java
- Lua
- Node
- Perl
- PHP
- Python
- Ruby
- Scala

## Configuration

Set an environment variable `ENV['GITHUB_API_TOKEN']` for accessing the Github API and not hitting the free rate limit.
